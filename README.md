# Fetchy

Fetchy is a lightweight Windows desktop application built with C# and .NET.
It provides a simple update checker and tool downloader, mainly designed
to fetch and update UnlockTool and related components.

The goal of Fetchy is to make downloading and updating tools easier,
faster, and more user-friendly.

--------------------------------------------------

Features

- Update checker to detect newer versions
- Tool downloader for UnlockTool and required files
- Simple Windows Forms based user interface

--------------------------------------------------

Built With

- C#
- .NET Framework
- Windows Forms

--------------------------------------------------

Getting Started

Clone the repository:

git clone https://github.com/Ctellar/Fetchy.git

Build and Run:

1. Open Fetchy.sln using Visual Studio
2. Restore dependencies if required
3. Build the solution
4. Run the application

--------------------------------------------------

Project Structure

- Fetchy.sln        : Main solution file
- Fetchy.csproj     : Project configuration
- Program.cs        : Application entry point
- Form1.cs          : Main UI logic
- icons/            : Application icons and assets
